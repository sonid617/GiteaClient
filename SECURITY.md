# Security Policy

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, use GitHub's private vulnerability reporting:
[Report a vulnerability](../../security/advisories/new)

I'll respond within 48 hours and work with you on a fix before public disclosure.

## Scope

- Auth token handling / Keychain storage
- API request security (HTTPS enforcement, header exposure)
- Any data leakage between Gitea accounts
